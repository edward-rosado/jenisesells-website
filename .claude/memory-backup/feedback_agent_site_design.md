## Tailwind v4 Font Override Problem

Tailwind v4's `@import "tailwindcss"` includes Preflight which resets `font-family: sans-serif` on all elements. The production site uses `body { font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif; }` with no Tailwind.

### What DIDN'T work (tried all of these):
1. Setting `--default-font-family` in `@layer theme` — Tailwind's own theme layer overrides it
2. Applying `font-family` to `html, body, h1, h2, ...` explicitly — Preflight still wins
3. Removing Preflight via `@import "tailwindcss/theme"` + `@import "tailwindcss/utilities"` — theme layer still injects font variables
4. Removing theme entirely (`@import "tailwindcss/utilities"` only) — still didn't stick

### What WORKED:
```css
@import "tailwindcss/utilities" layer(utilities);

* {
  font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif !important;
}
```
The `!important` on a universal selector is the nuclear option but it's the only thing that overrides Tailwind v4's internal font handling.

## Config Registry is Auto-Generated

`apps/agent-site/lib/config-registry.ts` is AUTO-GENERATED by `scripts/generate-config-registry.mjs`. It's gitignored.

**Critical rules:**
- Changes to `config/agents/*.json` are NOT reflected until regenerated: `node apps/agent-site/scripts/generate-config-registry.mjs`
- CI must run the generate script BEFORE tests (vitest doesn't trigger npm prebuild)
- The `prebuild` npm script runs it automatically before `next build`, but NOT before `vitest run`

## All Sections Use Inline Styles

Every section component (Hero, Nav, Footer, StatsBar, Services, HowItWorks, SoldHomes, Testimonials, About, CmaForm) uses `style={{}}` props matching the production CSS exactly. This avoids Tailwind conflicts entirely.

Exception: CmaForm and LegalPageLayout use some Tailwind utility classes for layout convenience.

## Production Site Reference (jenisesellsnj.com)

When matching production, always fetch the actual site HTML/CSS rather than guessing. Key details:
- Font: `'Segoe UI', Tahoma, Geneva, Verdana, sans-serif` (system fonts, no Google Fonts)
- Nav: Fixed top, green background (#1B5E20), gold accent (#C8A951)
- Phone icon: Inline SVG with `display: inline` (Tailwind Preflight sets SVGs to `display: block`)
- Office phone: In a gold pill button with `whiteSpace: nowrap`
- Footer: "Licensed Real Estate Salesperson | NJ License #..." and "Independent Agent with [brokerage]" on separate lines
- Service areas: "Serving Middlesex, Monmouth & Ocean Counties, NJ" (strip "County" from each, join with "&")
- Equal Housing Opportunity: Standalone centered icon+text, ABOVE the disclaimer paragraph
- Legal links (Privacy, Terms, Accessibility): Required for compliance, rendered in footer
