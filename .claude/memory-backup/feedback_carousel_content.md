# Carousel Content Requirements

**Extracted:** 2026-03-17
**Context:** User reviewed template previews and found a carousel showing only 1 photo

## Rule
Always add more than 1 item to any carousel, slideshow, or gallery component in test/demo data. A carousel with a single slide looks broken and defeats the purpose of the component.

## Why
Eddie reported "The luxury carousel only has 1 picture. We need to make sure we can cycle through the photos." Single-item carousels give a bad impression during demos and hide navigation bugs.
