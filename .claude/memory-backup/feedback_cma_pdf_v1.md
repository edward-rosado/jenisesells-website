# CMA PDF v1 Feedback (2026-03-27)

After reviewing the first branded CMA PDF:

1. **Logo too small and unreadable** — needs to be much bigger with a white background behind it (the green header band makes the logo hard to see)
2. **Header too large** — takes too much vertical space, pushing content off the page
3. **Footer too large** — same issue
4. **Content doesn't fit one page** — the comp table + analysis + pricing gets cut off or pushed to page 2 because header/footer consume too much space

**How to apply:**
- Reduce header height: smaller agent name font, tighter padding, keep logo readable
- Logo: larger with white background/padding box behind it
- Reduce footer: single line with essentials (name, phone, license, disclaimer)
- Target: all content fits on one US Letter page for a 5-comp report
- Test with actual Jenise config + 5 comps to verify fit
