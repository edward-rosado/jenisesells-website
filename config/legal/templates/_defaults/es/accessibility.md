## Estado de conformidad

Nuestro objetivo es cumplir con las **Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1 Nivel AA**. Estas pautas explican cómo hacer que el contenido web sea más accesible para personas con discapacidades.

## Medidas que tomamos

Para apoyar la accesibilidad, tomamos las siguientes medidas:
- Incluimos la accesibilidad como parte del diseño de nuestro sitio web y las decisiones de contenido
- Usamos HTML semántico para transmitir estructura y significado
- Proporcionamos alternativas de texto para contenido que no es de texto
- Garantizamos un contraste de color suficiente entre el texto y los fondos
- Hacemos que toda la funcionalidad esté disponible mediante navegación con teclado
- Evitamos contenido que pueda provocar convulsiones o reacciones físicas
- Proporcionamos navegación clara y consistente
